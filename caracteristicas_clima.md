# Características actuales de ClimaBot

- Consulta del clima actual usando `/weather <ciudad>`
- Pronóstico extendido con `/forecast [ciudad]`
- Gestión de ciudad favorita con `/setciudad` y `/ahora`
- Emoji automático según estado del tiempo
- Manejo de errores y respuestas claras al usuario
- Almacenamiento de configuración por usuario
- Comando `/about` con información del proyecto y autor
- Comando `/version` para saber la versión instalada
- Integración con archivo `.env` para manejo de secretos
- Desplegable como servicio con `systemd`
